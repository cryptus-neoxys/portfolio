# Dev Sharma's Portfolio (cryptus-neoxys)

This is a personalized portfolio website built with Next.js, showcasing my professional experience, skills, and projects.

## Setup
1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/cryptus-neoxys/portfolio.git
   ```

2. Navigate to the project directory:

   ```bash
   cd portfolio
   ```

3. Install dependencies:

   ```bash
   pnpm install
   ```

## Development

1. Start the development server:

   ```bash
   pnpm dev
   ```

2. Open http://localhost:3000 in your browser to view the site.

3. Edit the configuration in `src/data/resume.tsx` to customize content.

4. For production build:

   ```bash
   pnpm build
   ```
